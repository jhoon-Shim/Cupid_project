# 💌 사랑이 싹트는 42
### *터틀봇이 전하는, 차마 하지 못했던 말*

---

## 한 줄 요약

> **42 클러스터 실시간 좌석 맵을 기반으로, 터틀봇이 익명의 메시지를 직접 배달하는 오프라인 대나무숲**

---

## 프로젝트 배경

42 클러스터는 매일 수십 명의 학생이 모여 코딩하는 공간이다.  
하지만 그 삭막한 컴퓨터 화면 사이로, **차마 전하지 못한 말들**이 조용히 쌓여간다.

> *"저 사람한테 말 걸고 싶었는데…"*  
> *"오늘 도움받았는데 고맙다는 말을 못 했어."*  
> *"사실 좋아해."*

우리는 그 말들에게 **다리**를 놓아준다.  
터틀봇이 직접 찾아가서.

---

## 작동 방식

```
[발신자]
사용자가 웹/터미널 UI에서
→ 수신자 좌석 선택
→ 메시지 작성
→ 전송

        ↓

[터틀봇]
클러스터 맵 기반 자율주행으로
→ 해당 좌석까지 직접 이동

        ↓

[수신자 노트북]
터틀봇 도착과 동시에
→ 사랑의 BGM 재생 🎵
→ 감성 터미널 화면에 메시지 출력 💻
```

---

## 시스템 구조

| 구성요소 | 역할 | 비고 |
|---|---|---|
| **터틀봇** | 자율주행 메시지 배달 | ROS2 Navigation |
| **클러스터 맵** | 좌석 위치 매핑 | 가상 좌표 기반 |
| **발신자 UI** | 메시지 입력 & 전송 | 웹 or 터미널 |
| **수신자 노트북** | BGM + 감성 디스플레이 | 터미널 감성 출력 |

> 터틀봇에는 디스플레이/스피커가 없기 때문에,  
> **소리와 화면은 수신자의 노트북**이 담당한다.

---

## 핵심 기술

- **ROS2 Navigation2** — 터틀봇 자율주행 & 목표지점 설정
- **좌석 매핑** — 클러스터 실제 레이아웃 기반 가상 좌표 설계
- **노드 분리 구조** — 터틀봇(Nav 노드) ↔ 노트북(Display/Sound 노드) 소켓 통신
- **감성 터미널 UI** — 클로드 코드풍 터미널 + 2000년대 초 삐삐·싸이월드 감성

---

```
│  보낸 사람 : ??? (익명)                  │

│  " 오늘 옆에 앉아서 공부했는데           │

│                                         │

│  🎵 나는 너를  (BGM 재생 중...)          │
└─────────────────────────────────────────┘
```

**레퍼런스 무드:** 클로드 코드 터미널 + 싸이월드 방명록 + 2000년대 초 삐삐 문자
